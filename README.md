# IGDB Games

App to list games from IGDB Api using Bloc, GetIt, Dio and Floor.

[API Reference](https://api-docs.igdb.com/)

## Demo

![](demo/demo.gif)

## Features

  ✅ Pull to refresh.\
  ✅ Search loading debounced.\
  ✅ Infinity scroll pagination.\
  ✅ Offline persistence.

## Getting Started

Create a .env file with your data.

`flutter run`

To run tests

`flutter test`
